# API Reference

Retrieve form submissions programmatically using the FormBold REST API. This API allows you to integrate your form data into your own systems, build custom dashboards, or automate internal workflows.

> [!IMPORTANT]
> **Authentication Required:** All API endpoints require an API key. You can generate and manage your API keys from the **Account -> API Tokens** section in your FormBold dashboard.

---

## Authentication

All API requests must include your API token in the `Authorization` header as a Bearer token:

```bash
Authorization: Bearer YOUR_API_TOKEN
```

---

## List Submissions API

Retrieve, search, and paginate through all submissions for a specific form.

### Endpoint

`GET https://api.formbold.com/v1/submissions/{formId}`

### Parameters

| Name | Type | Description |
| :--- | :--- | :--- |
| `formId` | `string` | **Required.** The unique ID of your form. |
| `page` | `integer` | Optional. The page number to retrieve. Default is `1`. |
| `limit` | `integer` | Optional. Number of results per page. Default is `20`. |
| `search` | `string` | Optional. Search term to filter submissions by content. |

### Request Example

<Tabs items={['cURL', 'Node.js', 'Python']}>
  <Tabs.Tab>
```bash
curl -X GET "https://api.formbold.com/v1/submissions/YOUR_FORM_ID?page=1&limit=10" \
     -H "Authorization: Bearer YOUR_API_TOKEN"
```
  </Tabs.Tab>
  <Tabs.Tab>
```javascript
const response = await fetch('https://api.formbold.com/v1/submissions/YOUR_FORM_ID', {
  headers: {
    'Authorization': 'Bearer YOUR_API_TOKEN'
  }
});
const data = await response.json();
console.log(data);
```
  </Tabs.Tab>
  <Tabs.Tab>
```python
import requests

url = "https://api.formbold.com/v1/submissions/YOUR_FORM_ID"
headers = {"Authorization": "Bearer YOUR_API_TOKEN"}

response = requests.get(url, headers=headers)
print(response.json())
```

## Get Single Submission

Retrieve the full details of a specific submission.

### Endpoint

`GET https://api.formbold.com/v1/submissions/{formId}/{submissionId}`

---

## Delete Submission

Permanently delete a submission from your records.

### Endpoint

`DELETE https://api.formbold.com/v1/submissions/{formId}/{submissionId}`

---

## Error Responses

| Code | Description |
| :--- | :--- |
| `400` | **Bad Request:** The request was invalid or missing required parameters. |
| `401` | **Unauthorized:** Your API token is invalid or missing. |
| `404` | **Not Found:** The form or submission ID does not exist. |
| `429` | **Too Many Requests:** You have exceeded the rate limit. |
